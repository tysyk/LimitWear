# LimitWear

Платформа для запуску та продажу лімітованого одягу через систему дропів та попередніх замовлень.

## Про проєкт

LimitWear — це маркетплейс для дизайнерів, брендів та покупців, який дозволяє запускати лімітовані колекції одягу у форматі дропів.

На відміну від класичних інтернет-магазинів, виробництво може запускатися лише після підтвердження попиту через систему попередніх замовлень.

Це дозволяє мінімізувати ризики перевиробництва та допомагає дизайнерам перевіряти інтерес до своїх колекцій ще до запуску виробництва.

## Основні ролі

### Покупець

- перегляд дропів;
- оформлення замовлень;
- відстеження статусу замовлення;
- історія покупок.

### Дизайнер

- створення дизайнів;
- керування власними дропами;
- перегляд статистики продажів.

### Адміністратор

- модерація контенту;
- керування користувачами;
- керування дропами;
- контроль роботи платформи.

## Основні можливості

- система дропів;
- лімітовані колекції;
- попередні замовлення;
- особистий кабінет;
- адміністративна панель;
- модерація дизайнів;
- система ролей;
- аналітика продажів;
- підтримка брендів та незалежних дизайнерів.

## Концепція платформи

1. Дизайнер створює дизайн.
2. Дизайн проходить перевірку.
3. Створюється дроп.
4. Покупці оформлюють замовлення.
5. Після досягнення необхідних умов запускається виробництво.
6. Замовлення переходять на етап виконання та доставки.

## Статус проєкту

Проєкт знаходиться в активній розробці.

На даному етапі формується архітектура платформи, MVP функціонал та дизайн-система для подальшого масштабування.
